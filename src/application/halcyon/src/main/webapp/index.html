About Halcyon?

Halcyon is a RESTful web application designed to run in an OSGi container providing a cutting edge user interface to various hbird applications.

Three Core Principles

Skeleton is built on three core principles:

  • RESTful: Employs Restful web service principles.
  • OSGi: Real modular software. Plug and play different components at runtime.
  • Metro styling: Style inspiration
