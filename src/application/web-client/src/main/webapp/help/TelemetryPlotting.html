This page allows you to plot multiple live or historical parameters over time.

The view's default state is to plot live telemetry.

  • Add parameters using the search bar.
  • Remove currently there is no remove parameter function; it's en-route, sorry!
  • Historical values can be plotted by using the time filter controls and clicking the plot button. Parameters must be added to the chart first!
  • Zoom drag an area of the plot to zoom in. Double click to reset the zoom level. You can also use the small overview chart to zoom the main plot!
